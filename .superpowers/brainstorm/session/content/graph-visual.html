连线设计：粒子流光线，表示关系

Edge 视觉规范
active — 实线蓝色，粒子流动速度快
ended — 虚线灰色，粒子静态
conflict — 红色虚线，脉冲警示
连线粗细 = 1 + confidence × 3 px
Label 文字沿连线中间悬浮显示
evidence_count 越多 → 粒子密度越高

图谱布局与相机

A

3D 力导向布局 ★推荐

节点自然分散，高关联度节点聚拢；OrbitControls 拖拽旋转/缩放/平移；自动缓慢旋转（可选关闭）

B

径向/层级布局

中心人物/话题放射状展开，适合展示"谁和谁讨论了什么"的层级结构

支持一键切换两种布局，力导向为默认

跨模块联动行为

点击节点 → 触发联动
单击节点 Tooltip 弹出 → 显示节点详情 → 可选"查看聊天"/"聚焦"
双击节点 侧边栏高亮匹配的联系人 → 跳转对应聊天 → 图谱聚焦该节点
点击连线 显示关系详情（predicate, confidence, evidence_count）
搜索联动 全局搜索关键词 → 图谱中匹配的节点高亮闪烁 → 其他节点半透明
